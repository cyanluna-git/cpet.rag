## STEP 2. Colab에 키 등록
- [ ] https://colab.research.google.com 접속
- [ ] 새 노트북 아무거나 열고 → 왼쪽 **🔑(열쇠) 아이콘** 클릭
- [ ] **"새 보안 비밀 추가"** →
  - 이름: `JINA_API_KEY` / 값: 복사한 Jina 키 / **노트북 액세스 토글 ON**
  - (선택) 이름: `GEMINI_API_KEY` / 값: Gemini 키 / 토글 ON

## STEP 5. 셀 1~6 순서대로 실행  (▶ 버튼 또는 Shift+Enter)

- [ ] **셀 5** (Drive 마운트) — 팝업 뜨면 **본인 구글 계정 인증/허용**

### 빠른 문제해결
| 증상 | 해결 |
|------|------|
| `SecretNotFoundError` | (이미 수정됨) 노트북 다시 열거나 클론 셀 재실행 |
| 셀 6 PDF 0개 / 경로오류 | 셀 5 `PDF_DIR` 경로 수정 |
| Jina 401 | `JINA_API_KEY` 값 오타 확인 |
| 설치 너무 느림 | 정상 — 최초 1회만 오래 걸림 |
| 클론 실패 | repo가 public인지 확인 |

> 요약: **Jina 키 등록 → 노트북 열기 → GPU 켜기 → 셀 위→아래로 실행 → 7·8번 결과 가져오기.**
